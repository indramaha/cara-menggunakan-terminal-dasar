Comand Pada GIT

  1. pada saat membuka terminal pada laptop, kita berada pada home directory. untuk masuk ke dalam salah satu folder yang ada pada home directory kita, ketikkan "cd Documents" pada terminal, maka yang awalnya berada pada home, skarang pindah ke "home/Documents"
  2. untuk kembali dari "home/Documents", siahkkan ketikkan cd + spasi + titik dua kali "cd .." pada terminal, makan yang tadinya kita berada pada folder Documents, skarang sudah kembali lagi ke home
  3. untuk membuat folder baru, ketikkan "mkdir (nama folder)" pada terminal
  4. untuk membuat file, ketikkan "touch (nama file) pada terminal"
  5. menghapus file/ folder, ketikkan "rm -rf (nama file/folder)"
